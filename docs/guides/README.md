# Guides

Operational documentation for workspace users. These guides help you set up, customize, and maintain your workspace.

## Audience

These guides are for **satellite workspace users** — people who created their workspace using the dev-workspace-template and need to:

- Complete initial setup
- Understand the relationship between their workspace and the template
- Keep their workspace in sync with template improvements

## Not Covered Here

- **Template development** — See [CONTRIBUTING.md](../../CONTRIBUTING.md) in the template repo
- **Project-specific docs** — Belong in individual project repos under `repos/`
- **AI/Copilot configuration** — See [.github/](.github/) directory
